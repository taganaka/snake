# Snake
SDL2 powered retro snake game

My first deeep dive into casual gaming programming using C++ and SDL2

## How to compile

### On Mac

```
$ brew install sdl2 sdl2_image sdl2_ttf
$ cd build
$ cmake ..
$ make
$ ../bin/sdl_snake
```
